Streaming Client

This is a simulation of a number of users interacting with the streaming client installed on their devices (smart TVs, tablets, etc).

Each user can choose to stream one of the available videos, for a maximum of 3 concurrent streams per user.

The active stream is represented by an item in a list. This is a demo and there is streaming API or content to be consumed.
